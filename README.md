# Recipe Finder

Simple React app to search recipes using TheMealDB API.  
Features:

- Search recipes by keyword
- View recipe details (ingredients & instructions)
- Save favorites (localStorage)
- Responsive, minimal UI

## Tech

React, Vite, React Router, Fetch API

## Run locally

1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## Deploy

Works well on Vercel or Netlify. Build with `npm run build`.

## Notes

API used: https://www.themealdb.com/api.php (no API key required for search).
